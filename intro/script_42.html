bufexplorer.zip : Buffer Explorer / Browser

作者：jeff lanzarotta

脚本类型：公用(utility)

描述

使用bufexplorer，你可以快速、轻松的使用如下公开接口在多个缓冲区之间进行切换跳转：

  • '\be' （普通的打开方式）或者
  • '\bs' （强制在水平分栏窗口中打开）或者
  • '\bv' （强制是垂直分类窗口中打开）

一旦打开了缓冲区窗口，你就可以使用普通的跳转快捷键来进行移动，还可以使用 <Enter> 或 <Left-Mouse-Click> 来选择你希望打开的缓冲区。如果你喜欢在新页签(tab)中打开，简单的使用 <Shift-Enter> 或者 't' 就可以了。记住，当缓冲区在新页签中打开时，如果这个缓冲区也存在于其他页签中，那么只要你喜欢， bufexplorer 也可以自动切换到那个页签中。更多的信息请参考 Vim 帮助。

bufexplorer 提供了多种选项，包括：

  • 显示各种排序的缓冲区列表，排序方式包括：
    • 默认是最近常用(MRU)
    • 缓冲区序号
    • 文件名
    • 文件扩展
    • 文件的完整路径
  • 从缓冲区列表中删除

更多的选项，排序方式，配置选项，等等，请参考 Vim 帮助。

安装细节

解压缩 bufexplorer.zip 到你的 "runtimepath" 目录，通常是 ~/.vim 或者 c:\vimfiles ，并重启 Vim。压缩文件包含了 plugin\bufexplorer.vim 和 doc\bufexplorer.txt。请参考 ":help add-local-help" 来添加 bufexplorer.txt 到 Vim 的帮助系统中。

注意：插件的 7.0.12 版以上的只支持 Vim 的 7.0 版以上。

警告：如果你已安装了低于 7.12 版的插件（包含一个 autoload\bufexplorer.vim 文件，请先删除它，再将新版的 bufexplorer.vim 放至 plugin\bufexplorer.vim ）
