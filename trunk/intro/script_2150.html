css_color.vim : CSS 颜色预览

作者：Niklas Hofer

脚本类型：语法(syntax)

描述

这个脚本使用实际的颜色来衬垫CSS的十六进制颜色代码。并且使用恰当的前景色（文本颜色）。所以 #FF0000 会看起来像火一样。

这是预览图
http://lanpartei.de/~niklas/vim/after/syntax/css-color-vim.png

安装细节

将 css.vim 下载并放至 .vim/after/syntax/css.vim 即可。

备注：因为命名的颜色值白色(white)会将CSS属性 white-space 的white部分背景色显示为白色，影响正常的CSS语法高亮，而白色又比较容易识别记忆，所以可以注释掉第163行，或者直接下载这个 css.vim 。
