ActionScript.vim: ActionScript 3.0 syntax file

作者：Richard Leider

脚本类型：语法(syntax)

描述

更新 Igor Dvorsky 的 ActionScript 1.0 语法文件 vimscript#413 到包含 ActionScript 2.0 和 ActionScript 3.0 的关键字。

安装细节

ActionScript 文件的后缀是 ".as"，但是 Vim 的 filetype.vim 中关联的默认类型是 associated。你应该删除（或注释掉）association 关联并创建新的关联到 ActionScript。
