Vimbuddy : 一个绝对没有用的小玩意

作者：Flemming Madsen

脚本类型：公用(utility)

描述

在状态栏显示动态表情。在左右移动时，表情的鼻子会顺时针或逆时针轮转，发生错误或警告时还会皱眉头（译注：另外还有一些非常有趣有用的状态表情）。

安装细节

从 vimrc 中引入 vimbuddy.vim 或者将其放入 plugin/ 目录中，然后将 "%{VimBuddy()}" 放到状态栏( statusline )中。例如： 

set statusline=%{VimBuddy()}

更多帮助可以参考： 

:help statusline
