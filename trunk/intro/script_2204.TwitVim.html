作者：Po Shan Cheah

脚本类型：公用(utility)

描述

TwitVim 是一个 Vim 插件，让你可以在 Vim 中发推或者查看好友的推， 这个插件是基于 Travis Jeffery 开发的 vimscript#2124 的增强版。 荣誉属于 Travis ，属于他的创意，和最初的脚本实现。

TwitVim 拥有许多经典 Twitter 客户端所支持的特性，包括：

  • 好友，用户，私信，和时间线(timeline)
  • Twitter 搜索
  • 回复和锐推(RT)
  • 哈希标签(hashtags) （调整到搜索时间线）
  • 被回复的上下文
  • 在浏览器中打开网址
  • 显示用户资料
  • Twitter 列表(List)
  • URL 短地址 及 展开

TwitVim 使用 cURL 来跟 Twitter 通信，或者，也可以配置 TwitVim 来使用 Vim 的 Perl, Python, Ruby 或者 Tcl 接口这些更快的网络读写方式。

用法

:PosttoTwitter - 这个命令将提示你输入一条消息，来发送到 Twitter。 :CPosttoTwitter - 这个命令直接发送当前缓冲区当前行的内容到 Twitter。 :BPosttoTwitter - 这个命令直接发送当前缓冲区的内容到 Twitter。

Alt-T - 在可视选择模式下，<Alt-t> 快捷键直接发送选中的文本到 Twitter。 (如果启用了菜单栏，则使用 <Ctrl-t>)

注意：为了避免重写 T 搜索命令，T 被改为映射到 Alt-t 了。

Alt-R - 从时间线缓冲区开始一个 @-回复。 Alt-D - 从时间线缓冲区开始一个私信。

:FriendsTwitter - 查看好友的时间线。 :UserTwitter - 查看自己的时间线。 :RepliesTwitter - 查看 @-回复。 :PublicTwitter - 查看公开的随机时间线。 :DMTwitter - 查看私信。

:Tweetburner - 用 Tweetburner 缩短一个 URL 地址并插入到光标处。 :ATweetburner - 同上，但插入到光标之后。 :PTweetburner - 同上，但将返回的短地址放在命令栏，并开始准备发推。

以及一些类似的命令，如 TinyURL, SnipURL, Metamark, bit.ly, is.gd, urlBorg, tr.im, Cligs 和 zi.ma。

:SearchTwitter - 使用 Twitter 搜索。

更多的命令和键盘映射，可以参考 TwitVim 的帮助文档。

安装细节

新版都是使用 Vimball 档，打开 vba 文件并引用(:source)即可。 

vim twitvim.vba
:source %

更进一步的说明可以看 :help TwitVim-install。

下面是针对老版本(非 vba)的说明：

  1. 首先，你需要先安装 cURL，可以从 这里 下载。
  2. 然后添加脚本到 Vim 的 plugins 目录。实际位置是平台相关的，详细可以看 :help add-plugin。
  3. 再把下面这行加到 .vimrc 或 _vimrc 中： 
    4. 


    let twitvim_login = "USER:PASS"
    }}}
    修改 USER 和 PASS 为你的 Twitter 用户名和密码。
# 如果你通过代理上网，加入下面这行：
    {{{class="code vim"
    let twitvim_proxy = "proxyhost:proxyport"
    }}}
    修改 proxyhost 和 proxyport 为你的代理服务器地址和端口号。
